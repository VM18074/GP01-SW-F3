Tipos de Arreglos

Los arreglos en C# también conocidos como Arrays al igual que en la mayoria de lenguajes de programación orientados a objetos, son indexados iniciando en cero (0). La forma de trabajo es muy similar a la mayoría de lenguajes pero hay algunas diferencias que notarán.
Cuando vayamos a declarar un arreglo en debemos colocar los corchetes después de tipo de dato. 


                //Declaración valida
                int[] valores;

En C# podemos indicar el tamaño del arreglo luego de la declaración. Esto nos permite decidir que tamaño va a tener el arreglo según lo que necesitemos.
En C# tambien se pueden declarar matriz, es muy similar al arreglo pero con doble corchete[], ademas los vectores o matriz se pueden crear independiente de su tipo de dato, a continuacion un par de ejemplos: 


            //Arreglo definiendo su tamaño. 
            int[] valores; //valores sin inicializar
            valores = new int[100];  //100 elementos  

            //Arreglo de arreglos
            int[][] matriz;                   //sin inicializar
            //Los arreglos de arreglos se inicializan de manera diferente
            int[][] matriz = new int[3][];
                for (int i = 0; i < matriz.Length; i++)
                {
                    matriz[i] = new int[4];
                }




Sentencia Condicional if


La sentencia condicional if se usa para tomar decisiones, este evaluá básicamente una operación lógica, es decir una expresión que de como resultado Verdadero o Falso, y ejecuta la pieza de código siguiente siempre y cuando el resultado sea verdadero.

La sentencia if EXPRESION, significa, Si se cumple la expresión condicional se ejecuta el bloque de sentencias seguidas. 


                
                if (edad <= 18) //Condición.
                {
                    Console.WriteLine("Es un adolescente."); 
                    //Se ejecutara si la condición es verdadera. 
                }




Sentencia Condicional if-else


A la instrucción if se le puede agregar una cláusula else, que permite ejecutar una instrucción o un bloque de instrucciones en el caso de que la condición no se cumpla.

En la siguiente imagen definimos la condició, esta como sabemos es de tipo booleano, que significa que solo posee dos resultados, verdadero representadop por el bloque 1, y falso representado por el bloque2, lo que significa que dependiendo del resultado de la condición sera el bloque de código que se va a ejecutar, en este tipo de sentencias es imposible que se ejecuten los dos bloques de código. 


                if (edad <= 18)
                {
                    Console.WriteLine("Es un adolescente."); 
                    //Se ejecutara si la condición es verdadera. 
                }
                else
                {
                    Console.WriteLine("No es un adolescente."); 
                    //Se ejecutara si la condición es falsa
                }





Sentencia Condicional if-else if-else


También conocido conomo if anidados. O condiciones anidadas. Dentro del bloque de instrucciones podemos incluir cualquier instrucción. Es muy común en programacion encontrar sentencias if dentro de sentencias if.

Lo que se quiere representar es que se puede tener un if dentro de un if o else, a lo que se concluye es que despues de tener el resultado veradero o falso, dentro del código a ejecuatar se puede colocar otro if, esto en la programación funciona para validar en varios puntos un dato. 



                if (lado1 == lado2 && lado2 == lado3) {
                    JOptionPane.showMessageDialog(null, "El triángulo es equilátero");
                } else {
                    if (lado1 == lado2 ||  lado2 == lado3 || lado1 == lado3) {
                        JOptionPane.showMessageDialog(null, "El triángulo es isósceles");
                    } else {
                        JOptionPane.showMessageDialog(null, "El triángulo es escaleno");
                    }
                }

                //También se puede representar de la siguiente manera
                if (lado1 == lado2 && lado2 == lado3)
                {
                    JOptionPane.showMessageDialog(null, "El triángulo es equilátero");
                }
                else if (lado1 == lado2 ||  lado2 == lado3 || lado1 == lado3)
                {
                    JOptionPane.showMessageDialog(null, "El triángulo es isósceles");
                }
                else
                {
                    JOptionPane.showMessageDialog(null, "El triángulo es escaleno");
                }
