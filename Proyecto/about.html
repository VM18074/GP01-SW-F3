Sobre el proyecto

Se pretende Implementar sitio web de contenido propio para dar a conocer el lenguaje de programación (historia, origen, área de aplicación del lenguaje de programación: C#, características, etc. enfocado a enseñar de manera efectiva pero en líneas generales el uso de las estructuras de control, estructuras de datos básicas, de la programación), las intenciones del sitio web será enseñar de una manera breve las características del lenguaje de programación a base de ejemplos, con la cual se tiene la idea de crear un cuadro de texto con la sintaxis con la opcion de copiar fácilmente las lineas de codigo, para que el usuario pueda tener una interacción continua con la web, nos expandimos a las herramientas con ejemplos online(compiladores online), desktop y si hay móvil.


Sobre nosotros

Misión

Contribuir al desarrollo tecnológico e informático del país a través de experiencias educativas, eficientes y efectivas para la formación de futuros programadores.


Visión

Desbloquear el potencial de cada persona, para que descubran y desarrollen el programador que llevan dentro.
